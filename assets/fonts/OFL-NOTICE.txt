Bundled display fonts and their licenses
========================================

Fonts bundled for burned-in typography (Pillow/drawtext), all under the SIL
Open Font License 1.1 (OFL-1.1), which permits bundling and redistribution.

- Inter-Bold.ttf        — Inter, The Inter Project Authors
                          https://github.com/rsms/inter   (see Inter-LICENSE.txt)
- Montserrat-Bold.ttf   — Montserrat, The Montserrat Project Authors
                          https://github.com/JulietaUla/Montserrat
- Teko-Bold.ttf         — Teko, Indian Type Foundry
                          https://github.com/google/fonts/tree/main/ofl/teko

Full OFL-1.1 text: https://openfontlicense.org/

Optional extra weights can be fetched with: python scripts/download_fonts.py
