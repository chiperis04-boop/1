# Assets

Drop your channel assets here. None are committed (keep them out of git if licensed).

```
assets/
├── music/          # royalty-free background tracks (.mp3/.wav). Round-robin per clip.
├── sfx/
│   └── crowd_roar.wav   # one-shot crowd swell mixed on the key beat
├── fonts/
│   └── Inter-Bold.ttf   # caption + overlay font (any .ttf works)
└── branding/
    ├── intro.mp4   # 0.8–1.5s logo sting (optional; text card used if missing)
    └── outro.mp4   # CTA sting (optional)
```

## Where to get free, license-clean assets

- **Music:** Pixabay Music, Free Music Archive (check the licence per track).
- **SFX (crowd):** freesound.org (CC0/CC-BY), Pixabay SFX.
- **Fonts:** Google Fonts (Inter, Montserrat, Anton are good for sports captions).

> Replace the placeholder paths in `config/config.yaml` and `config/branding.yaml`
> if you use different filenames.
